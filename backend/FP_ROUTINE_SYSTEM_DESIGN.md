# FP 루틴 기반 고객 접점 관리 시스템

**작성일**: 2025-12-05
**핵심 철학**: "성실한 루틴이 신뢰를 만들고, 신뢰가 성과를 만든다"

---

## 📋 핵심 개념

### FP 성공 공식
```
성공 = 루틴의 일관성 × 접점의 질 × 시스템의 자동화

루틴의 일관성: 매일, 매주, 매월 정해진 활동 반복
접점의 질: 단순 연락 → 가치 제공 (정보, 도움, 관심)
시스템의 자동화: 사람은 실행만, 계획/추적/알림은 시스템이
```

### 고객 관계 단계별 접점 전략

```
┌─────────────────────────────────────────────────────────┐
│           고객 관계 라이프사이클 (6단계)                 │
└─────────────────────────────────────────────────────────┘

1단계: 인지 (Awareness) - 첫 만남
   📱 문자: 명함 교환 후 당일 감사 메시지
   ⏱️ 타이밍: 만남 후 3시간 이내

2단계: 관심 (Interest) - 니즈 파악
   ☎️ 전화: 1주일 후 안부 + 간단한 보험 현황 질문
   ⏱️ 타이밍: 첫 만남 후 5-7일

3단계: 평가 (Evaluation) - 제안 단계
   🤝 미팅: 보장 분석 결과 대면 설명
   📊 자료: DOCX 리포트, 비교표
   ⏱️ 타이밍: 전화 후 3-5일

4단계: 결정 (Decision) - 계약 체결
   🤝 미팅: 계약서 작성
   📱 문자: 계약 감사 + 서비스 안내
   ⏱️ 타이밍: 제안 후 1-2주

5단계: 유지 (Retention) - 관계 지속
   📱 문자: 월 1회 정기 안부
   ☎️ 전화: 분기 1회 보험 리뷰 제안
   🤝 미팅: 연 1-2회 대면
   ⏱️ 타이밍: 평생

6단계: 옹호 (Advocacy) - 추천 고객 확보
   🎁 이벤트: VIP 고객 초대
   📱 문자: "주변 분께 소개해주세요" 요청
   ⏱️ 타이밍: 관계 만족도 높을 때
```

```python
class CustomerPriorityCalculator:
    """
    고객 우선순위 자동 계산
    """

    def calculate_priority_score(self, customer: Customer) -> float:
        """
        우선순위 점수 계산 (0-100)
        """
        score = 0

        # 1. 계약 가능성 (40점)
        if customer.status == "proposal_sent":
            days_since = (date.today() - customer.last_proposal_date).days
            if days_since <= 7:
                score += 40  # 1주일 이내
            elif days_since <= 14:
                score += 30  # 2주일 이내
            elif days_since <= 30:
                score += 20  # 1개월 이내

        # 2. 관심도 (30점)
        if customer.interest_level == "high":
            score += 30
        elif customer.interest_level == "medium":
            score += 20
        elif customer.interest_level == "low":
            score += 10

        # 3. 타이밍 (20점)
        # 생일, 갱신, 라이프 이벤트
        if self.is_birthday_soon(customer):
            score += 20
        elif self.is_renewal_soon(customer):
            score += 20
        elif self.has_lifecycle_event(customer):
            score += 15

        # 4. 잠재 가치 (10점)
        # 가족 규모, 소득 수준
        if customer.family_size >= 3:
            score += 5
        if customer.income_level == "high":
            score += 5

        return min(score, 100)

    def get_priority_customers(self, fp_id: str, limit: int = 20) -> List[Customer]:
        """
        우선순위 상위 고객 추출
        """
        customers = db.query(Customer).filter(
            Customer.fp_id == fp_id
        ).all()

        # 우선순위 점수 계산
        scored_customers = [
            {
                "customer": customer,
                "score": self.calculate_priority_score(customer),
                "reason": self.get_priority_reason(customer)
            }
            for customer in customers
        ]

        # 점수 순 정렬
        sorted_customers = sorted(
            scored_customers,
            key=lambda x: x["score"],
            reverse=True
        )

        return sorted_customers[:limit]
```
